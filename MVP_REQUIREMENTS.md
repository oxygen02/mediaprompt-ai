# PromptBox AI - MVP v1.0 需求文档

## 项目概述

PromptBox AI 是一款内容逆向工程工具，帮助用户从文档、图片、视频和网站中提取AI提示词，进而生成类似内容。

**核心定位：**
- 用户上传想要分析的内容 → AI 按维度分析并给出提示词 → 用户编辑提示词 → 生成相似内容

---

## 用户旅程

```
1. 上传模仿对象（文档/图片/视频/网页）
   ↓
2. 选择分析模型（系统自动推荐或用户自选）
   ↓
3. 点击"开始分析"
   ↓
4. AI 按选中的维度进行逆向分析
   ↓
5. 输出整合后的提示词（填充到"提示词再创作"区域）
   ↓
6. 用户编辑提示词（可选：上传源文件）
   ↓
7. 选择生成模型，点击"创意类似+"
   ↓
8. 分享/下载生成结果
```

## 功能模块

### 1. 左侧导航栏

**需求：**

- 显示 PromptBox Logo + 标题
- 分类导航：文档分析、图片分析、视频分析、网页分析、文档案例、历史记录、设置
- 当前选中项高亮显示
- 底部显示用户信息

**交互动画：**
- 无（文字和图标始终显示）

### 2. 顶部导航栏

**需求：**
- 固定高度 56px
- 左侧：首页、定价、关于链接
- 中间：PromptBox Logo + 标题
- 右侧：语言切换（中文/EN）、登录、注册

### 3. 顶部副标题区域

**需求：**
- 固定位置（top: 56px）
- 副标题文字："发现AI生成内容的创作秘诀"
- 四类内容切换按钮（月球图标 + 文字）
- 选中项高亮显示

**交互动画：**
- 选中时月球图标旋转 180 度

### 4. 上传模仿对象区域

**需求：**
- 支持拖放上传
- 显示文件类型提示（根据当前选中的内容类型）
- 鼠标悬停左上角显示"上传模仿内容"标题
- 底部显示提示词选项（可多选）

**支持格式：**
- 文档：Word/PDF/TXT/Markdown
- 图片：JPG/PNG/WebP/GIF
- 视频：MP4/MOV（≤30秒）
- 网页：URL/网页截图

**提示词选项（根据内容类型动态显示）：**

| 内容类型 | 分析维度 |
|---------|---------|
| 文档 | 核心主题、文案体裁、语言风格、语气情感、内容结构、字数长度、目标受众、使用场景、核心关键词 |
| 图片 | 画面主体、创作风格、构图视角、光影效果、色彩搭配、画质细节、背景环境、整体氛围 |
| 视频 | 画面主体、镜头运镜、视频风格、光影色调、画质帧率、节奏时长、内容逻辑、氛围情绪 |
| 网页 | 网页类型、设计风格、主色搭配、页面布局、功能模块、适配设备、整体氛围、细节质感 |

### 5. AI 分析模块

**需求：**
- 模型选择下拉框（上下文感知）
- 开始分析按钮
- AI 思考过程折叠显示（10步）
- 分析结果填充到提示词再创作区域

**分析模型推荐：**

| 内容类型 | 中文界面推荐 | 英文界面推荐 |
|---------|-------------|-------------|
| 文档 | 混元Lite(免费) > 通义千问(免费) > GPT-4o | GPT-4o > Claude 3.5 > Gemini |
| 图片 | 通义千问VL(免费) > 混元视觉(免费) > GPT-4o Vision | GPT-4o Vision > Claude Vision > Gemini |
| 视频 | 通义千问VL(免费) > 混元视觉(免费) > GPT-4o Vision | GPT-4o Vision > Gemini |
| 网页 | 混元Lite(免费) > 通义千问(免费) > GPT-4o | GPT-4o > Claude 3.5 > Gemini |

### 6. 提示词再创作区域

**需求：**
- 可编辑 textarea
- AI 分析结果自动填充
- 鼠标悬停左上角显示"创作你自己的提示词"标题
- 有内容时隐藏悬停标题，避免文字重叠

**提示词输出格式：**
```
【核心主题】XXX
【文案体裁】XXX
【语言风格】XXX
...

━━━━ 整合提示词 ━━━━
创作一篇关于XXX的文章，主题聚焦XXX。语言风格XXX，语气XXX...
```

| 文本 | 混元Pro > 通义千问Max > GPT-4o | GPT-4o > Claude 3.5 > 混元Pro |
| 图片 | Midjourney > DALL-E 3 > Stable Diffusion > Flux(免费) | Midjourney > DALL-E 3 > Stable Diffusion |
| 视频 | 可灵AI > Sora > Runway > Pika | Sora > Runway > Pika > 可灵AI |
| 网页 | V0 > Cursor > Bolt.new > Trae(免费) | V0 > Cursor > Bolt.new > Replit |

### 8. 视频专属选项

**需求（仅在视频类型时显示）：**

## API 接口需求

### 1. 内容分析接口
```
POST /api/analyze
Content-Type: multipart/form-data

Request:
{
  file: File,           // 上传的文件
  contentType: string,  // document | image | video | website
  model: string,        // 选择的分析模型
  options: string[]     // 选中的分析维度
}

Response:
{
  success: boolean,
  result: string,       // 整合后的提示词
  steps: string[]       // 分析过程各步骤结果
}
```

### 2. 内容生成接口
```
POST /api/generate
Content-Type: multipart/form-data

Request:
{
  prompt: string,       // 提示词
  sourceFile: File,     // 可选的源文件
  contentType: string,  // 生成内容类型
  model: string,        // 选择的生成模型
  videoSize: string,    // 视频尺寸（视频类型时）
  voiceover: string     // 配音选项（视频类型时）
}

Response:
{
  success: boolean,
  result: string        // 生成结果或生成链接
}
```
