# MediaPrompt AI - 产品需求文档 v2

## 一、核心定位

**一句话**：上传你喜欢的内容，AI 帮你分析、拆解，输出可复制的提示词、代码和工具推荐。

**目标用户**：非专业人士，零编程基础，不知道用什么工具、不知道给 AI 什么提示词。

**核心价值**：像反向工程一样，把优秀内容拆解成可执行的操作指南。

---

## 二、四种输入类型与输出

### 1. 文字文档分析

**输入**：Word/PDF/TXT/Markdown，或直接粘贴文字
- 商业文案、公文、社交媒体文案、小说、剧本、策划书等

**输出**：
| 输出项 | 说明 |
|--------|------|
| 结构分析 | 大纲、段落逻辑、信息层次 |
| 风格分析 | 用词特点、句式结构、语气调性 |
| 仿写提示词 | 可直接用于 AI 写作工具的提示词 |
| 工具推荐 | ChatGPT、Claude、文心一言、通义千问等 |

**用户场景**：
- 看到一篇好的商业策划书，想仿写一份
- 看到一个爆款文案，想写出类似风格
- 看到一段小说描写，想学习这种笔法

---

### 2. 图片分析

**输入**：JPG/PNG/WebP/GIF，或图片链接
- AI 生成的图、照片、海报、UI 界面等

**输出**：
| 输出项 | 说明 |
|--------|------|
| 风格分析 | 艺术风格、视觉调性、情感氛围 |
| 配色方案 | 主色、辅色、色值、配色比例 |
| 构图分析 | 布局、焦点、透视、比例 |
| 提示词 | Midjourney / Stable Diffusion / DALL-E 可用提示词 |
| 代码（UI类） | Tailwind CSS 代码（针对 UI 界面图） |
| 工具推荐 | 推荐最适合的 AI 绘图工具和模型 |

**用户场景**：
- 看到一张好看的 AI 图，想生成类似风格
- 看到一张海报，想自己做一张类似的
- 看到一个 UI 界面，想学习这种设计风格

---

### 3. 视频分析

**输入**：视频文件（≤30秒），或视频链接
- 短视频、广告、产品展示、教程、Vlog、AE 特效等

**输出**（用户选择输出目标）：
| 输出目标 | 说明 |
|----------|------|
| 一键同风格提示词 | 复刻该视频风格的完整提示词 |
| 调整建议 | 分析可改进点，给出优化方案 |
| AI 工具推荐 | 适合生成此类视频的工具和模型 |

**风格标签**：电影感、Vlog、广告、产品展示、教程、短视频、动画、AE 特效

**用户场景**：
- 看到一个酷炫的视频效果，想知道怎么做的
- 看到一个产品展示视频，想做类似的
- 看到一个爆款短视频，想分析它的节奏和风格

---

### 4. 网页逆向分析（核心功能）

**输入**：网页 URL

**输出**：

#### 4.1 基础分析
| 输出项 | 说明 |
|--------|------|
| 功能特色 | 网站主要功能和亮点 |
| 技术栈 | 使用的前端框架、UI 库、后端技术 |
| API 分析 | 调用的第三方服务和接口 |
| 大模型 | 如果有用到 AI，分析可能使用的模型 |
| 需求文档 | MVP 需求文档模板，可直接给开发者 |

#### 4.2 设计分析
| 输出项 | 说明 |
|--------|------|
| 整体风格 | 设计语言、视觉调性 |
| 布局结构 | 页面骨架、信息架构 |
| 配色方案 | 主色、辅色、色值 |
| 字体方案 | 标题字体、正文字体、字号层级 |
| 组件库 | 可能使用的 UI 组件库 |

#### 4.3 提示词与代码
| 输出项 | 说明 |
|--------|------|
| 生成提示词 | 让 AI 帮你创建类似网页的提示词 |
| Tailwind 代码 | 核心布局和组件的可复制代码 |
| HTML/CSS 片段 | 关键区块的代码 |

#### 4.4 浮层标注（创新功能）
**功能描述**：在原网页上叠加一个交互式标注层

**交互方式**：
1. 用户输入 URL 后，系统加载该网页（代理或截图）
2. 用户鼠标悬停在任何元素上
3. 显示浮层信息：
   - 元素类型（按钮/卡片/标题等）
   - 尺寸大小（宽×高）
   - 位置信息（margin/padding）
   - 颜色值（背景色、文字色、边框色）
   - 字体信息（字体、字号、字重、行高）
   - CSS 类名（如果是 Tailwind）
4. 一键复制该元素的样式代码
5. 一键生成该元素的提示词

## 三、界面设计（学习 YouWare）

### 设计原则
1. **极简上传区** — 首屏就是一个大输入框/上传区
2. **引导式创建** — What → Who → Goal → Style 步骤引导
3. **实时预览** — 左侧输入，右侧实时展示结果
4. **卡片流结果** — 多个方案以卡片形式展示，可滑块选择数量（2-6个）
5. **一键复制** — 所有输出都有一键复制按钮

## 四、记忆系统

### 4.1 快速启动
- 新用户选择 20 个预设风格包（极简、科技、复古、国风等）
- 建立初步偏好画像

### 4.2 持续学习
- 用户划选批注 → 自动提取规则
- 手动添加个性化指令
- 达到 20 条规则后生成个人风格画像

### 4.3 风格画像
- 可视化雷达图（配色偏好、风格倾向、复杂度等）
- 时序演变图（偏好变化趋势）
- JSON 导出导入（跨设备同步）

## 八、与 YouWare 的差异

| 维度 | YouWare | MediaPrompt AI |
|------|---------|----------------|
| 核心功能 | AI 生成网站 | AI 分析内容，输出提示词 |
| 输入 | 文字描述 + 图片 | 文字/图片/视频/网页 |
| 输出 | 可发布的网站 | 提示词 + 代码 + 工具推荐 |
| 目标用户 | 想做网站的人 | 想用 AI 创作但不会写提示词的人 |
| 特色功能 | 实时预览网站 | 网页浮层标注 |

**定位**：YouWare 是「正着做」（描述→网站），我们是「反着来」（内容→提示词→工具）。

## 九、总结

MediaPrompt AI 是一个**反向工程式的内容分析工具**，帮助小白用户：
1. 上传喜欢的内容
2. AI 分析拆解
3. 输出可复制的提示词、代码、工具推荐
4. 用户用这些结果，在其他 AI 工具中生成想要的内容

**核心理念**：不会写提示词？那就上传你喜欢的，AI 帮你写。
