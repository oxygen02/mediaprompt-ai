## 一、产品概述

### 1.1 产品名称
PromptBox AI - 媒体提示词智能分析平台

### 1.2 产品定位
一款基于AI的媒体内容分析工具，将文档、图片、视频、网页转化为可复用的AI提示词。

### 1.3 核心功能
- 多媒体类型支持（文档、图片、视频、网页）
- AI智能分析与提示词生成
- 中英文双语界面
- 结构化输出展示

## 四、输出格式规范

### 4.1 结构化提示词格式

```markdown
## Role
[角色定义]

## Background
[背景信息]

## Profile
[简介]

## Skills
- [技能1]
- [技能2]

## Goals
- [目标1]
- [目标2]

## Constraints
- [约束1]
- [约束2]

## OutputFormat
[输出格式说明]

## Workflow
1. [步骤1]
2. [步骤2]

## Examples
[示例]

## Initialization
[初始化配置]
```

### 4.2 视频分析输出维度

1. **提示词整合** - 完整提示词
2. **场景描述** - 环境、背景、空间布局
3. **风格** - 色调、滤镜、画面质感
4. **光影** - 光源方向、明暗对比、氛围
5. **主体与动作** - 人物/物体、行为、表情
6. **镜头变化** - 运镜方式、景别、转场
7. **分镜说明** - 时长分配、节奏感、情绪曲线
8. **细节分析** - 道具、文字、特效、音效
9. **动态细节** - 动作流畅度、动态效果

## 五、API接口规范

### 5.1 视频分析接口

**POST /api/analyze/video**

请求：
```json
{
  "videoUrl": "https://...",
  "options": ["场景描述", "风格", "光影"],
  "aspect": "16:9",
  "voiceover": "专业解说风格"
}
```

响应：
```json
{
  "success": true,
  "data": {
    "thinking": [
      { "step": "parse", "text": "解析视频文件..." },
      { "step": "extract", "text": "提取关键帧..." }
    ],
    "result": {
      "prompt": "完整提示词",
      "blocks": {
        "scene": "场景描述内容",
        "style": "风格描述内容"
      }
    }
  }
}
```

### 5.2 视频压缩接口

**POST /api/video/compress**

请求：
```json
{
  "videoUrl": "https://...",
  "maxSize": 10485760
}
```

响应：
```json
{
  "success": true,
  "data": {
    "compressedUrl": "https://...",
    "originalSize": 52428800,
    "compressedSize": 8388608
  }
}
```
