## 一、产品概述

### 1.1 产品定位
MediaPrompt AI 是一款面向非技术用户的内容反向工程工具，帮助用户从已有的优质内容中提炼有效的 AI 提示词，实现"让美好可以复刻迭代"。

### 1.2 目标用户
- **主要用户**：不懂提示词的小白用户（内容创作者、运营人员、设计师）
- **次要用户**：需要快速生成提示词的专业用户

### 1.3 核心价值
- **学习效率**：从模仿开始，快速掌握优质内容的创作逻辑
- **降低门槛**：无需深入学习提示词工程，一键生成专业提示词
- **可复用性**：生成的提示词可用于多种 AI 模型和工具

## 二、MVP 功能范围

### 2.1 核心功能（P0 - 必须有）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| 文档分析 | 上传文档（Word/PDF/TXT/Markdown），AI 分析并生成结构化提示词 | P0 |
| 图片分析 | 上传图片，AI 分析并生成 AI 绘画提示词 | P0 |
| 提示词输出 | 结构化输出（Role/Background/Profile/Skills/Goals/Constrains/OutputFormat/Workflow/Examples/Initialization） | P0 |
| 模型选择 | 支持选择不同 AI 模型进行分析 | P0 |
| 一键复制 | 复制生成的提示词 | P0 |

### 2.2 重要功能（P1 - 应该有）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|
| 视频分析 | 上传短视频（≤30秒），生成视频创作提示词 | P1 |
| 网页分析 | 输入 URL 或截图，分析网页设计并生成提示词 | P1 |
| 案例展示 | 展示同一段提示词在不同模型下的输出对比 | P1 |
| 编辑修改 | 用户可手动编辑生成的提示词 | P1 |
| 模拟预览 | 使用提示词生成示例输出 | P1 |

### 2.3 后续功能（P2 - 可以有）

| 功能模块 | 功能描述 | 优先级 |
|----------|----------|--------|

## 三、用户故事

### 3.1 文档分析场景

**用户故事 1：短视频运营学习**
> 作为一名短视频运营新手，我想分析热门短视频的运营策略文档，以便学习专业运营的思路和方法。

**验收标准：**
- 用户可以上传 Word/PDF 格式的运营文档
- 系统生成结构化的运营策略提示词
- 提示词包含 Role、Background、Skills 等完整字段
- 用户可以复制提示词用于 ChatGPT/Claude 等工具

**用户故事 2：产品需求文档学习**
> 作为一名产品经理新人，我想分析优秀产品的 PRD 文档，以便学习专业产品经理的文档撰写思路。

**验收标准：**
- 支持 Markdown 格式的 PRD 文档上传
- 系统识别文档类型并生成对应的提示词模板
- 输出包含需求分析、用户故事、验收标准等结构

### 3.2 图片分析场景

**用户故事 3：AI 绘画提示词生成**
> 作为一名 AI 绘画爱好者，我想上传喜欢的图片，获取可用于 Midjourney/SD 的提示词。

**验收标准：**
- 支持 JPG/PNG/WebP/GIF 格式
- 系统分析图片的主体、风格、构图、光影、色彩
- 输出 Midjourney/SD/DALL-E 兼容的提示词
- 提供参数建议（分辨率、采样器等）

### 3.3 案例展示场景

**用户故事 4：模型输出对比**
> 作为一名内容创作者，我想查看同一段提示词在不同 AI 模型下的输出效果，以便选择最适合的模型。

**验收标准：**
- 展示文档、图片、视频、网页四类案例
- 每个案例展示 3+ 种模型的输出对比
- 标注模型名称、风格特点
- 支持分类筛选

### 5.1 分析请求

```json
{
  "content": "用户输入或文件内容",
  "category": "document|image|video|website",
  "outputOptions": ["prompt", "outline", "structure", "keywords"],
  "model": "hunyuan-lite|hunyuan-pro|gpt-4o|claude-3.5"
```

## 六、定价策略

### 6.1 免费版
- 每日 10 次分析
- 支持文档、图片分析
- 基础模型（混元 Lite）

### 6.2 专业版（¥39/月）
- 无限次分析
- 全部分析类型（文档/图片/视频/网页）
- 高级模型（混元 Pro、GPT-4o）
- 历史记录保存
- 优先客服支持

### 6.3 企业版（定制）
- API 接入
- 私有化部署
- 定制化功能
- 专属客服

## 八、风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| AI 模型成本过高 | 财务压力 | 使用混元 Lite 免费额度，限制免费用户次数 |
| 用户上传敏感内容 | 合规风险 | 内容审核机制，用户协议明确责任 |
| 竞品模仿 | 市场竞争 | 快速迭代，积累优质案例库 |
| 服务不稳定 | 用户体验 | 多模型备份，服务监控告警 |
