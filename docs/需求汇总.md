# PromptBox AI - 需求汇总

**版本：** V35 修复版  
**日期：** 2026-03-18  
**产品名称：** PromptBox AI（媒体提示词智能分析平台）

---

## 一、产品定位

一款基于AI的媒体内容分析工具，能够从文档、图片、视频、网页中提取有效提示词，让用户可以将优质内容"复刻"到自己的创作中。

**核心价值：** 把看到的好内容，变成可以复用的AI提示词。

---

## 二、目标用户

1. **AI创作者** - 需要将参考内容转化为AI提示词
2. **内容运营** - 分析竞品内容的提示词结构
3. **设计师/摄影师** - 将视觉作品转化为Midjourney/Stable Diffusion提示词
4. **视频创作者** - 提取视频风格、场景、运镜等信息

---

## 三、支持的内容类型

### 1. 文档 (Document)
- 支持格式：Word/PDF/TXT/Markdown
- 输出选项：提示词整合、功能列表、MVP文档、技术栈

### 2. 图片 (Image)
- 支持格式：JPG/PNG/WebP/GIF
- 输出选项：提示词整合、场景描述、风格设定、光线与氛围

### 3. 视频 (Video) ⭐核心功能
- 支持格式：MP4/MOV（建议30秒内）
- 输出选项：提示词整合、场景描述、风格、光影、主体与动作、镜头变化、分镜说明、细节分析、动态细节
- **专属选项：**
  - 视频尺寸选择：16:9 / 9:16 / 4:3 / 1:1 / 21:9
  - 同期AI配音：横条文本输入框

### 4. 网页 (Website)
- 支持格式：URL/网页截图
- 输出选项：提示词整合、功能列表、MVP文档

---

## 四、界面功能要求

### 1. 语言切换
- 中文 / English
- 完整界面切换（所有文案、按钮、提示）
- 语言状态持久化

### 2. 社交媒体分享
- **中文界面：** 微信、抖音、小红书、微博
- **英文界面：** X (Twitter)、Facebook、LinkedIn、Instagram
- 样式：灰色圆角图标按钮，与整体风格协调

### 3. 下载功能
- 一键导出分析结果为 .txt 文件
- 复制到剪贴板功能

### 4. 预览功能
- 标题："创意类似+" / "Similar Ideas+"
- 流式输出展示
- AI思考过程折叠区（10步）

---

## 五、交互设计要求

### 1. 按钮交互
- 所有按钮必须可点击
- 使用 onclick 属性直接绑定（避免 addEventListener 问题）
- 点击后有视觉反馈

### 2. 类别切换
- 点击切换时高亮当前选中
- 视频类别必须显示专属选项区域

### 3. 选项标签
- 支持多选（点击切换选中状态）
- 选中状态视觉区分

### 4. 视频尺寸按钮
- 选中状态高亮
- 互斥选择（单选）

---

## 六、技术要求

### 1. 前端
- 原型：HTML + Tailwind CSS
- 最终：Next.js 16 + Tailwind CSS 4 + TypeScript

### 2. 后端
- Node.js + Express
- 腾讯云混元 API
- 千问 VL API (qwen3-vl-plus)

### 3. 视频处理
- 必须先压缩再发送给 VL API
- 支持获取视频元数据
