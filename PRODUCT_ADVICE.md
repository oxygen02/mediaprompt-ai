# MediaPrompt AI 产品建议书

## 一、一句话定位

**让小白用户上传喜欢的内容，一键获得「能直接用的」AI提示词和工具推荐**

---

## 二、核心差异化（3个关键词）

| 关键词 | 说明 |
|--------|------|
| **零门槛** | 不需要懂AI，不需要会写提示词 |
| **可复制** | 输出结果直接可用，不是理论分析 |
| **工具链** | 不只给提示词，还告诉你用什么工具、怎么用 |

---

## 三、目标用户

**主力用户**：想做内容但不会用AI的小白
- 想发小红书但写不出文案的素人
- 想做视频但不会写脚本的新手
- 想做海报但不懂设计的店主

**次要用户**：想提高效率的内容创作者
- 已经有基础，但想快速复刻爆款风格

---

## 四、MVP功能清单

### 必须做（P0）

| 功能 | 说明 |
|------|------|
| 文字分析 | 上传文案/文章，输出仿写提示词 |
| 图片分析 | 上传图片，输出生成提示词（Midjourney/SD） |
| 一键复制 | 所有提示词可一键复制到剪贴板 |
| 工具推荐 | 推荐适合的AI工具，带跳转链接 |

### 稍后做（P1）

| 功能 | 说明 |
|------|------|
| 网页分析 | 分析网页风格，输出提示词 |
| 历史记录 | 保存分析记录，方便回看 |
| 收藏功能 | 收藏常用提示词 |

### 暂不做（P2）

| 功能 | 原因 |
|------|------|
| 视频分析 | 技术复杂、成本高、需求低频 |
| 代码输出 | 小白用户用不上 |
| 浮层标注 | 开发成本高、使用场景窄 |

---

## 五、商业模式

### 定价策略

| 方案 | 价格 | 权益 |
|------|------|------|
| 免费版 | ¥0 | 每天5次，仅文字+图片 |
| 专业版 | ¥39/月 | 每天50次，全部功能 |
| 团队版 | ¥199/月 | 无限次+多人协作 |

### 收入测算（假设）

- 日活用户：1000人
- 付费转化率：5%
- 月活付费用户：50人 × ¥39 = ¥1950/月

**初期目标**：覆盖成本，验证PMF

---

## 六、关键成功指标

| 指标 | 目标值 | 验证什么 |
|------|--------|----------|
| 日活/注册 | >30% | 用户是否愿意回来用 |
| 分析完成率 | >70% | 流程是否顺畅 |
| 复制率 | >50% | 输出是否有用 |
| 付费转化率 | >3% | 用户是否愿意付费 |

---

## 七、技术方案

### 选型

| 组件 | 方案 | 理由 |
|------|------|------|
| 前端 | Next.js + Tailwind | 快速开发、SEO友好 |
| 后端 | Node.js + PostgreSQL | 简单够用 |
| AI模型 | 腾讯Coding API | 成本低、中文好 |
| 存储 | 腾讯云COS | 国内访问快 |

### 成本估算

| 项目 | 月成本 |
|------|--------|
| 服务器 | ¥200 |
| AI API | ¥500（按1000日活估算） |
| 存储/CDN | ¥100 |
| **合计** | **¥800/月** |

---

## 八、风险与应对

| 风险 | 应对 |
|------|------|
| AI输出质量不稳定 | 多模型交叉验证+人工审核 |
| 用户上传违规内容 | 接入内容审核API |
| 竞品快速复制 | 聚焦用户体验，建立品牌认知 |
| 成本失控 | 设置每日调用上限 |

---

## 九、下一步行动

1. **本周**：确定UI设计，完成前端静态页面
2. **下周**：接入AI API，实现文字+图片分析
3. **第3周**：内测，收集反馈
4. **第4周**：修复问题，准备上线

---

## 十、总结

**产品价值**：让不懂AI的人也能用好AI

**核心策略**：先做减法，聚焦文字+图片分析，把体验做到极致

**成功标准**：用户觉得「比直接用ChatGPT方便10倍」
