## 核心流程

### 1. 分析流程

```
用户上传 → 格式检测 → 预处理 → 入队 → AI分析 → 结果生成 → 缓存 → 返回
   │                    │                    │
   ├─ 图片: 压缩/缩放    ├─ 图片: 视觉分析    ├─ 自然语言描述
   ├─ 视频: 帧提取      ├─ 视频: 分镜拆解    ├─ 提示词 (MJ/SD)
   ├─ 网页: 截图/抓取   ├─ 网页: 结构分析    ├─ Tailwind代码
   └─ 文档: 文本提取    └─ 文档: 风格提取    └─ 调整建议
```

### 2. 记忆系统流程

```
新用户 → 选择20个预设风格包 → 快速启动画像
    │
    ▼
使用积累 → 用户批注/选择 → 规则提取 → 添加到个人规则库
    │                              │
    ▼                              ▼
≥20条规则 → 生成个人风格画像 ← 持续优化
    │
    ├─ 雷达图可视化
    ├─ 时序演变图
    └─ JSON导出/导入 (跨设备同步)
```

## 技术栈选型

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| **前端** | Next.js 14 + Tailwind + TypeScript | App Router, RSC |
| **后端** | Node.js (NestJS/Fastify) | API 服务 |
| **视频处理** | FFmpeg + Python Worker | 帧提取、压缩 |
| **AI模型** | 腾讯 Coding 模型 API | 主模型，全功能 |
| **数据库** | PostgreSQL + PGVector | 主库 + 向量搜索 |
| **缓存** | Redis | 缓存 + 队列 + 会话 |
| **对象存储** | 腾讯云 COS | 文件存储 |
| **部署** | Docker + K8s / 腾讯云 | 弹性伸缩 |
| **支付** | Stripe | 国际支付 |
